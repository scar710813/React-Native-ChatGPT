# react-native-chatgpt3

Interact with ChatGPT3 directly from your React Native apps!

## Installation

```sh
npm install react-native-chatgpt3
```

## Usage

```js
import { multiply } from 'react-native-chatgpt3';

// ...

const result = await multiply(3, 7);
```
